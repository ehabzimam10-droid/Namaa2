# سجل تزامن الذكاء الاصطناعي (AI Sync Log) - ترحيل بيانات الأبناء سحابياً وتطوير الحصالة المتقدمة (v-14)

يوثق هذا السجل عملية ترحيل حسابات ومستندات الأبناء سحابياً إلى جداول قاعدة بيانات Supabase الحية، وبناء خوارزمية الفك والتنشيط التلقائي للأهداف الادخارية (المستهدفة والزمنية) للطفل.

## 📂 الملفات المشمولة بالتغيير

### 1. الملفات الجديدة:

### 2. الملفات المعدلة:
*   **[mockData.ts](file:///d:/Documents/Namaa/src/data/mockData.ts)**:
    *   تعديل واجهة `SavingsGoal` لإضافة المتغير الاختياري لتاريخ الاستحقاق الزمني:
        `deadlineDate?: string;` (بصيغة ISO).
*   **[AppContext.tsx](file:///d:/Documents/Namaa/src/context/AppContext.tsx)**:
    *   **ترحيل الحالة السحابية (Supabase Migration)**:
        *   أثناء تهيئة التطبيق (`fetchData`)، يتم قراءة جدول `kids_profiles` سحابياً وتعبئة قيم الرصيد المتاح والمدخرات الكلية ونقاط التبرع بشكل فوري بدلاً من استخدام البيانات المحلية المقلدة.
        *   تحديث الأفعال المالية `addDonation` و `investInProject` و `addToGoal` و `withdrawGoal` لتقوم بتنفيذ استعلام تعديل سحابي متزامن:
            `await supabase.from('kids_profiles').update({ balance: updatedBalance, ... }).eq('id', kidId);`
            لضمان التوافق التام والتزامن اللحظي للبيانات مع لوحة ولي الأمر.
    *   **منطق فك القفل التلقائي المتقدم (Advanced Auto-Unlock Logic)**:
        *   تطوير دالة تفقد الحصالات `checkSavingsStatus()` والتي تعمل تلقائياً مع تحميل الواجهات وعبر مؤقت زمني (كل 10 ثوانٍ).
        *   تقوم الدالة بتفقد جميع الحصالات المقفلة؛ فإذا تحقق الهدف المالي المطلوب (`currentAmount >= targetAmount`) **أو** انتهت المدة المحددة لتاريخ الاستحقاق (`deadlineDate` وتجاوز الوقت الفعلي للتطبيق التاريخ المسجل)، يتم تحرير الحصالة تلقائياً ومحو بطاقتها، مع نقل كامل المبالغ المستحقة لـ `balance` الطفل سحابياً ومحلياً، وتوليد عملية إيداع موثقة باسم **"استحقاق حصالة 💰"**.
*   **[KidSavingsPage.tsx](file:///d:/Documents/Namaa/src/pages/KidSavingsPage.tsx)**:
    *   تعديل نموذج تأسيس أهداف الادخار لإتاحة خانة إضافية اختيارية لتحديد تاريخ الاستحقاق: `تاريخ الاستحقاق (اختياري) 📅` من النوع `date`.
    *   عرض التواريخ بصورة زجاجية ومنظمة داخل تفاصيل بطاقة تقدم الحصالات لإثراء تجربة الطفل ومتابعة مدخراته.
