# سجل تزامن الذكاء الاصطناعي (AI Sync Log) - تهيئة وتكامل Supabase (v-4)

## 🗄️ هيكل وتخطيط جدول قاعدة البيانات (Database Schema - profiles)

يرتبط جدول التعريفات الشخصية بجدول المستخدمين الأصلي في Supabase Auth عبر مفتاح خارجي (Foreign Key) لربط الهوية بالبيانات المخصصة:

*   **اسم الجدول**: `profiles`
*   **الأعمدة والخصائص**:
    1.  `id`: نوع البيانات `uuid` (مفتاح أساسي Primary Key، يربط مباشرة بـ `auth.users(id)`).
    2.  `full_name`: نوع البيانات `text` (يمثل الاسم الفعلي المعروض في الواجهة مثل: "أبو خالد" أو "سالم").
    3.  `role`: نوع البيانات `text` (يمثل دور المستخدم في النظام مثل: `'father'` أو `'kid'` أو `'dev'`).
    4.  `created_at`: نوع البيانات `timestamp with time zone` (تاريخ الإنشاء، مع قيمة افتراضية للوقت الحالي تلقائياً).

1.  تأكد من وجود متغيرات البيئة المناسبة في ملف `.env` المحلي.
