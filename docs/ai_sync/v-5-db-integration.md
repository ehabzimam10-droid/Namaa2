# سجل تزامن الذكاء الاصطناعي (AI Sync Log) - تكامل قاعدة البيانات وسياق الحالة العامة (v-5)

تم بنجاح بناء ودمج محرك حالة عالمي (Global State Manager) يعتمد على سياق React (`React Context`) ومتزامن بالكامل مع التخزين المحلي (`LocalStorage`) ليعمل كـ "محرك قاعدة بيانات" متكامل ومستمر للتطبيق. يهدف هذا التحديث إلى استبدال البيانات الوهمية الساكنة بتدفق بيانات تفاعلي لحظي (Real-time Reactive Data Flow) يربط لوحة تحكم الأب بلوحة تحكم الأبناء مباشرة. 

يوثق هذا السجل كافة التفاصيل التقنية والبرمجية للتغييرات باللغة العربية لمساعدة المطورين القادمين.

## 📂 الملفات التي تم إنشاؤها وتعديلها

### 1. الملفات الجديدة:
*   **[v-5-db-integration.md](file:///d:/Documents/Namaa/docs/ai_sync/v-5-db-integration.md)**:
    *   **الوصف**: هذا المستند (سجل التزامن الحالي) باللغة العربية.
*   **[AppContext.tsx](file:///d:/Documents/Namaa/src/context/AppContext.tsx)**:
    *   **الوصف**: سياق الحالة العامة للتطبيق (`AppContext`) والوسيط المزود للحالة (`AppProvider`). 
    *   **المنطق**:
        *   تخزين الملف الشخصي الحالي للمستخدم (`profile`).
        *   تخزين مصفوفة الأطفال (`kids`) مع بياناتهم المالية والمهام الخاصة بكل منهم.
        *   تخزين مصفوفة المشاريع العائلية الاستثمارية (`projects`).
        *   مزامنة كافة الحالات السابقة تلقائياً مع `localStorage` لضمان بقائها حتى عند تحديث الصفحة أو إغلاق المتصفح، مع تهيئتها بقيم `mockData.ts` عند التشغيل الأول.
        *   توفير الدوال اللازمة للعمليات التفاعلية:
            *   `addDonation(kidId, amount)`: خصم مبلغ التبرع من مدخرات الطفل وزيادة نقاط التبرع، مع توليد معاملة سحب جديدة ديناميكياً لإدراجها في قائمة المعاملات الأخيرة للطفل.
            *   `approveTask(kidId, taskTitle, rewardAmount, rewardType, customReward)`: إضافة مهمة جديدة بحالة معلقة (`pending`) إلى مصفوفة مهام الطفل المعني.
            *   `addProject(title, totalRequired, roiPercentage)`: إضافة مشروع استثماري عائلي جديد بميزانية مستثمرة ابتدائية تبلغ (0 ريال).

        *   بناء قسم كامل وجديد لعرض "المهام والمسؤوليات الحالية" لقراءة وعرض المهام الموجهة للطفل وحالتها الحالية لحظة بلحظة.

## 🤖 معلومات هامة للعملاء القادمين (سياق العمل القادم):
1.  **تدفق التبرعات**: يؤدي التبرع بـ 10 ريال إلى خصم فوري من مدخرات الطفل الحالية (`saved`) مع زيادة نقاط التبرع وتوليد عملية سحب لتبقى الحسابات سليمة وتفاعلية.
2.  **استمرارية البيانات**: كافة البيانات تُحفظ في الـ LocalStorage تحت الأسماء: `namaa_profile`, `namaa_kids`, `namaa_projects`. لتصفير محرك البيانات والبدء من جديد، يمكن تصفير الـ localStorage من لوحة المطورين أو أدوات المتصفح.
