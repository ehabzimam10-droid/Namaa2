## 🎯 عن المشروع (الفكرة الأساسية)
**نماء** هي منصة عائلية تقنية متطورة، تهدف إلى بناء الوعي المالي (الادخار، الاستثمار، الاستهلاك، الخير) لدى الأطفال واليافعين، مع إعطاء الآباء لوحة تحكم كاملة للحوكمة والإشراف.
تتميز المنصة بكونها تخرج عن نطاق التطبيقات المالية التقليدية بتقديم **قرية ثلاثية الأبعاد (3D Village)** تتطور حرفياً وتنمو مبانيها (البنك، المزرعة، الطاحونة) بناءً على سلوك الطفل المالي الحقيقي، مدعومة بذكاء اصطناعي (Gemini) يوجه الطفل والأب.

---

## ✨ المميزات الرئيسية الفريدة
1. **القرية المالية التفاعلية (3D Gamification)**: واجهة مبنية بتقنية Three.js / React Three Fiber، تحاكي مدينة حية تتطور مبانيها كلما أنجز الطفل مهامه أو ادخر أمواله.
2. **متجر مكافآت شركاء الإنماء (B2B Rewards Store)**: نموذج عمل مبتكر يربط النقاط العائلية بشراكات حقيقية (مثل Sony PlayStation وJarir). يمكن للأطفال استبدال نقاطهم بأكواد وقسائم شراء حقيقية، مع الخصم التلقائي وحفظ تاريخ الاسترداد.
3. **المستشار الذكي (Gemini AI Coach)**: ذكاء اصطناعي مدمج يقوم بتحليل حالة الطفل المالية، ورصيده، وتطور قريته، ليقدم له نصائح باللغة العربية تشجعه على الادخار والاستثمار، وتوجيه الأبناء لكيفية استغلال نقاطهم واستبدالها من متجر شركاء الإنماء.
4. **حوكمة الأب (Father Dashboard)**: لوحة تحكم متكاملة تتيح للأب إضافة مهام، ربطها بمكافآت، إطلاق مشاريع استثمارية عائلية، وبدء دوريات وتحديات مالية بين الأبناء.
5. **النظام الهجين الآمن (Hybrid Sync)**: التطبيق مبرمج ليعمل بكفاءة 100% سواء كان متصلاً بقاعدة البيانات (Supabase) للعمل السحابي، أو عند انقطاع الاتصال (LocalStorage) كبيئة محلية مستقلة للتحكيم والتجربة الآمنة.

## ⚙️ الإعدادات التقنية (Configuration)

### 🔑 تشغيل ميزة الذكاء الاصطناعي (Gemini)
تم تجهيز التطبيق بنظام تنبيه ذكي. في حال عدم وجود مفتاح، سيعرض واجهة المستشار الذكي رسالة تفيد بأنه في (وضع المحاكاة). لتفعيل التحليل المباشر:
1. انقر على **زر الإعدادات السريعة (أيقونة المطور 👨‍💻)** الموجود في القائمة الجانبية أو الشريط السفلي.
2. أدخل مفتاح **Gemini API Key** الخاص بك.
3. انقر "حفظ"، وستعمل ميزات الذكاء الاصطناعي فوراً دون الحاجة لإعادة التشغيل!

### 🗄️ ربط قاعدة البيانات (Supabase)
إذا كنت تختبر التطبيق محلياً (Offline) أو بدون إعداد متغيرات البيئة، فإن التطبيق سيعمل بسلاسة عبر **LocalStorage** (حفظ محلي مؤقت). 
أما في حال أردت تفعيل المزامنة السحابية الكاملة، تأكد من إضافة المتغيرات التالية في ملف `.env`:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
